dht22
-----


## Compiling
Use the build.sh to compile the code

    $ ./build.sh 

## Running
Execute the dht22 binary.  Example:

    $ ./dht22 -p 25
    Program initialization
    setup event triggers
    humidity=-9999.00   temperature=-9999.00
    humidity=43.90   temperature=74.30
    humidity=43.90   temperature=74.30
